1.Clone the Repo (ek baar dono ko karna hoga):
git clone https://github.com/username/repo-name.git
cd repo-name


2.Create a Branch (har banda apna branch banaye):
git checkout -b my-feature-branch


3.Kaam karo apne branch mein (code changes etc.),Changes Stage karo and Commit:
git add .
git commit -m "Added new feature"


4.Push karo apne branch ko GitHub pe:
git push origin my-feature-branch


5.Hamesha latest changes pull karo before starting new work:
git checkout main
git pull origin main
